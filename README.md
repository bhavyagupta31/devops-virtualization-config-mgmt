# DevOps Virtualization and Configuration Management

**Student:** Bhavya Gupta
**Course:** DevOps — Virtualization & Configuration Management

---

## Overview

This repository contains all practicals completed as part of the DevOps course, covering containerization with Docker, multi-container orchestration with Docker Compose, CI/CD pipelines using GitHub Actions and Jenkins, and configuration management concepts.

---

## Repository Structure

```
practicals/
├── practical-01-docker-installation/       # Docker installation & verification
├── practical-02-docker-basics/             # Basic Docker commands
├── practical-03-dockerfile/                # Building custom images with Dockerfile
├── practical-04-docker-storage/            # Docker volumes & data persistence
├── practical-05-docker-compose-wordpress/  # WordPress + MySQL with Docker Compose
├── practical-06-docker-compose-fullstack/  # Full-stack app (Node + Java + PostgreSQL)
├── practical-07-docker-networking/         # Docker networking concepts
├── practical-08-github-actions/            # CI pipeline with GitHub Actions
└── practical-09-jenkins/                   # CI/CD pipeline with Jenkins
```
